<b>BCV PRICES BOT - @bcvpricesbot</b>

Este bot actualiza los precios del mercado de dólares venezolano a través de dolarapi.com.

Tiene varios comandos los cuales facilitan su uso, este bot puede calcular cuánto es en divisa cierta cantidad de bolívares, o cuanta cantidad de bolívares es cierta cantidad de divisa.

Pueden usarlo a través de @bcvpricesbot, este bot fue hosteado en render.com
